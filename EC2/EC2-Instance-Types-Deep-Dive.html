Selection Guide

Performance Requirements

  • • CPU-intensive: Choose Compute Optimized (C family)
  • • Memory-intensive: Choose Memory Optimized (R, X, z1d)
  • • Storage-intensive: Choose Storage Optimized (I, D, H)
  • • GPU workloads: Choose Accelerated Computing (P, G, F)

Cost Optimization

  • • Burstable: T instances for variable workloads
  • • Graviton2: a instances for better price/performance
  • • Previous generation: Lower cost for non-critical workloads

Network Performance

  • • Enhanced Networking: SR-IOV for higher PPS
  • • Placement Groups: Cluster for low latency
  • • Elastic Fabric Adapter: HPC workloads
