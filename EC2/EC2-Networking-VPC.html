Elastic Network Interfaces

Virtual network cards that can be attached to EC2 instances for flexible networking.

  • Primary ENI: Cannot be detached, created with instance
  • Secondary ENI: Can be attached/detached, moved between instances
  • Attributes: Private IP, Elastic IP, MAC address, Security Groups
  • Use Cases: Dual-homed instances, network appliances, licensing
# Create and attach ENI
aws ec2 create-network-interface \
  --subnet-id subnet-12345678 \
  --groups sg-12345678

aws ec2 attach-network-interface \
  --network-interface-id eni-12345678 \
  --instance-id i-12345678 \
  --device-index 1

Enhanced Networking

High-performance networking features for demanding applications.

SR-IOV

  • • Single Root I/O Virtualization
  • • Higher bandwidth, lower latency
  • • Higher packet per second (PPS) performance
  • • Available on most current generation instances

Elastic Fabric Adapter (EFA)

  • • High Performance Computing (HPC) workloads
  • • Bypass kernel for ultra-low latency
  • • MPI (Message Passing Interface) support
  • • Available on C5n, M5dn, M5n, R5dn, R5n instances

Placement Groups

Control instance placement to meet specific requirements for performance or availability.

Cluster

  • • Low latency, high network throughput
  • • Same Availability Zone, close physical proximity
  • • HPC applications, distributed databases
  • • Risk: Single point of failure

Spread

  • • Instances on distinct hardware
  • • Maximum 7 instances per AZ
  • • Critical applications requiring isolation
  • • Reduces risk of simultaneous failures

Partition

  • • Groups of instances in separate partitions
  • • Up to 7 partitions per AZ
  • • Large distributed systems (Hadoop, Cassandra)
  • • Balance between performance and availability

IP Addressing

Private IP Addresses

  • Assigned from subnet CIDR block
  • Remains with instance during stop/start
  • Used for internal VPC communication

Public IP Addresses

  • Dynamic, changes on stop/start
  • Only for instances in public subnets
  • Free when instance is running

Elastic IP Addresses

  • Static public IP address
  • Can be moved between instances
  • Charged when not associated

IPv6 Support

VPC supports dual-stack (IPv4 and IPv6). IPv6 addresses are globally unique and public by default.
