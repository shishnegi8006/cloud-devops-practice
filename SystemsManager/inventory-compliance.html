Inventory & Compliance

Comprehensive asset management and compliance monitoring

Inventory Management

Systems Manager Inventory collects metadata from your managed instances, providing visibility into system configurations, installed applications, and compliance status.

Inventory Types:

AWS:Application

Installed applications and packages

  • Application name and version
  • Installation date and path
  • Publisher information

AWS:InstanceInformation

Instance metadata and configuration

  • OS type and version
  • Hardware specifications
  • Network configuration

AWS:WindowsUpdate

Windows update status

  • Installed updates
  • Missing patches
  • Update history

Collection Schedule:

  • Default: Every 30 minutes
  • Customizable intervals
  • On-demand collection
  • Association-based scheduling

Compliance Monitoring

Monitor configuration compliance across your infrastructure using predefined and custom compliance rules.

Compliance Types:

Patch Compliance

  • Missing security patches
  • Critical update status
  • Patch baseline adherence

Association Compliance

  • State Manager association status
  • Configuration drift detection
  • Remediation tracking

Custom Compliance

  • Organization-specific rules
  • Third-party compliance frameworks
  • Custom validation scripts

Compliance Status:

  • Compliant: Meets all requirements
  • Non-Compliant: Fails one or more checks
  • Unspecified: No data available
