Questions & Answers

Practice questions

Question 1

Which Systems Manager capability provides browser-based shell access?

A) Run Command
B) Session Manager ✓
C) Parameter Store
D) Patch Manager

Correct: B) Session Manager provides secure browser-based shell access.

Question 2

What is the maximum number of parameters in Parameter Store Standard tier?

A) 10,000 ✓
B) 25,000
C) 50,000
D) 100,000

Correct: A) 10,000 parameters per region in Standard tier.

Question 3

Which parameter type in Parameter Store is encrypted using KMS?

A) String
B) StringList
C) SecureString ✓
D) All types

Correct: C) SecureString parameters are encrypted using KMS keys.

Question 4

What is required on EC2 instances to use Systems Manager capabilities?

A) SSM Agent ✓
B) CloudWatch Agent
C) AWS CLI
D) EC2 Instance Connect

Correct: A) SSM Agent must be installed and running on instances.

Question 5

Which Systems Manager feature is used for automated patching?

A) Run Command
B) Patch Manager ✓
C) Session Manager
D) Automation

Correct: B) Patch Manager handles automated patching workflows.

Question 6

What is the maximum number of concurrent Run Command executions?

A) 500
B) 1,000 ✓
C) 2,500
D) 5,000

Correct: B) 1,000 concurrent executions per region.

Question 7

Which feature collects system inventory and configuration data?

A) Parameter Store
B) Session Manager
C) Inventory ✓
D) Patch Manager

Correct: C) Inventory collects system and application data.

Question 8

What does Systems Manager Automation use to define workflows?

A) CloudFormation templates
B) Automation documents ✓
C) Lambda functions
D) Step Functions

Correct: B) Automation documents define multi-step workflows.
