Workflow automation and orchestration

Automation Documents

Define multi-step workflows using JSON or YAML automation documents that can orchestrate complex operational tasks.

Document Types:

  • Command: Execute commands on instances
  • Policy: Enforce configuration states
  • Automation: Multi-step workflows
  • Package: Software distribution

Pre-built Runbooks:

  • AWS-CreateImage (AMI creation)
  • AWS-RestartEC2Instance
  • AWS-StopEC2Instance
  • AWS-UpdateLinuxAmi
  • AWS-SetupInventory

Custom Workflows:

  • Multi-service orchestration
  • Conditional logic and branching
  • Error handling and rollback
  • Parameter passing between steps
