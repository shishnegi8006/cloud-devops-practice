Patch Manager

Automated patching at enterprise scale

Advanced Features

Automate the process of patching managed instances with security-related and other types of updates.

Patch Baselines:

  • Pre-defined: AWS-provided baselines
  • Custom: Organization-specific rules
  • Approval Rules: Auto-approve by classification
  • Patch Filters: Include/exclude specific patches

Maintenance Windows:

  • Scheduled patching windows
  • Recurring schedules (cron expressions)
  • Duration and cutoff settings
  • Notification integration

Compliance Reporting:

  • Patch compliance status
  • Missing patches identification
  • Compliance dashboards
  • Historical reporting

Patch Classifications

Security Patches

  • Critical security vulnerabilities
  • Security updates and hotfixes
  • CVE-addressed patches

Critical Updates

  • System stability fixes
  • Performance improvements
  • Bug fixes

Feature Updates

  • New functionality
  • Enhancement packages
  • Optional updates

Platform Support

  • Windows Server (all versions)
  • Amazon Linux, RHEL, Ubuntu
  • SUSE, CentOS, Debian

Testing Strategy
