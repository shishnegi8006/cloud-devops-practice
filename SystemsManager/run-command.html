Run Command

Remote command execution at scale

Advanced Capabilities

Execute commands remotely on multiple instances simultaneously with comprehensive control and monitoring.

Execution Features:

  • Multi-instance: Execute on thousands of instances
  • Rate Control: Limit concurrent executions
  • Error Handling: Stop on errors or continue
  • Output Capture: Collect and store command output
  • Timeout Control: Set execution time limits

Document Types:

  • AWS-RunShellScript (Linux/macOS)
  • AWS-RunPowerShellScript (Windows)
  • AWS-ConfigureAWSPackage
  • AWS-InstallApplication
  • Custom documents

Rate Control Configuration

  • Percentage-based: Execute on % of targets
  • Linear ramp: Gradual rollout strategy
