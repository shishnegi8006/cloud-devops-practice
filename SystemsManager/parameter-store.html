Parameter Store

Configuration management service

Parameter Store Overview

AWS Systems Manager Parameter Store provides secure, hierarchical storage for configuration data management and secrets management. It's a centralized store to manage your configuration data, whether plain-text data such as database strings or secrets such as passwords.

Key Features:

  • Hierarchical Storage: Organize parameters in tree-like structure
  • Versioning: Track parameter changes over time
  • Policies: Set expiration and notification policies
  • Integration: Native integration with AWS services
  • Cross-Region: Share parameters across regions

Pricing Tiers:

  • Standard: Free tier with basic features
  • Advanced: Enhanced features with charges

Parameter Types Deep Dive

String Parameters

Plain text configuration values

  • Database connection strings
  • Application configuration
  • Environment variables
  • Feature flags

StringList Parameters

Comma-separated values for lists

  • Server lists
  • Allowed IP addresses
  • Configuration arrays
  • Multi-value settings

SecureString Parameters

Encrypted using AWS KMS

  • Database passwords
  • API keys and tokens
  • License keys
  • Sensitive configuration

Advanced Features & Implementation

Hierarchical Organization

/myapp/dev/database/url

/myapp/prod/database/url

/myapp/test/database/url

  • Environment-based separation
  • Application-specific namespaces
  • Easy bulk operations
  • Inheritance patterns

Parameter Policies

  • Expiration: Auto-delete after specified time
  • ExpirationNotification: Alert before expiration
  • NoChangeNotification: Alert if not updated
  • Lifecycle Management: Automated cleanup
