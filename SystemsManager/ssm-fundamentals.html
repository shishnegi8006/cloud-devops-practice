SSM Fundamentals

Core concepts and capabilities

What is Systems Manager?

AWS Systems Manager is a comprehensive management service that provides a unified user interface to view operational data from multiple AWS services and automate operational tasks across your AWS resources.

Core Capabilities:

  • Operations Management: Centralized dashboard for resource health
  • Application Management: Parameter Store, AppConfig
  • Change Management: Automation, Change Calendar
  • Node Management: Session Manager, Run Command, Patch Manager

Key Benefits:

  • Reduce operational overhead
  • Improve security and compliance
  • Centralize operational data
  • Automate common tasks
  • No additional charges for most features

SSM Agent Deep Dive

The SSM Agent is Amazon software that runs on EC2 instances and on-premises servers to enable Systems Manager capabilities.

Installation & Availability:

  • Pre-installed: Amazon Linux 2, Ubuntu Server 16.04/18.04/20.04
  • Windows: Available on Windows Server 2008-2022
  • Manual Install: RHEL, CentOS, SUSE, Debian
  • Hybrid: On-premises servers and VMs

Communication:

  • HTTPS outbound on port 443
  • No inbound ports required
  • Uses AWS APIs for communication
  • Supports VPC endpoints

IAM Requirements:

  • AmazonSSMManagedInstanceCore policy
  • Instance profile or IAM role
  • Additional policies for specific features
