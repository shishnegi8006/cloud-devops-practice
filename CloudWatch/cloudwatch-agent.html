CloudWatch Agent

Enhanced System Monitoring

CloudWatch Agent Overview

The CloudWatch agent enables you to collect more system-level metrics from EC2 instances and on-premises servers. It can collect both system metrics and log files.

System Metrics

  • • Memory utilization
  • • Disk space utilization
  • • Swap utilization
  • • Network metrics
  • • Process metrics

Log Collection

  • • Application logs
  • • System logs
  • • Custom log files
  • • Windows event logs
  • • IIS logs

Configuration Example

{
  "metrics": {
    "namespace": "CWAgent",
    "metrics_collected": {
      "mem": {
        "measurement": ["mem_used_percent"]
      },
      "disk": {
        "measurement": ["used_percent"],
        "resources": ["*"]
      }
    }
  },
  "logs": {
    "logs_collected": {
      "files": {
        "collect_list": [
          {
            "file_path": "/var/log/messages",
            "log_group_name": "/aws/ec2/var/log/messages"
          }
        ]
      }
    }
  }
}

Key Benefits

Enhanced Monitoring

Collect system-level metrics not available by default, including memory and disk utilization.

Centralized Logging

Automatically send log files to CloudWatch Logs for centralized management and analysis.

Cross-Platform Support

Works on EC2 instances, on-premises servers, and supports both Linux and Windows.

Key Takeaways

  • CloudWatch Agent provides enhanced system monitoring beyond default EC2 metrics
  • Collects memory, disk, and custom metrics not available by default
  • Supports both metric collection and log file forwarding to CloudWatch
