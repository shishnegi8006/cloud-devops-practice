CloudWatch Events Overview

CloudWatch Events (now Amazon EventBridge) delivers a near real-time stream of system events that describe changes in AWS resources. You can create rules to match events and route them to targets for processing.

Key Takeaways

  • CloudWatch Events enables event-driven architectures and automation
  • Rules match events and route them to multiple targets
  • Supports both AWS service events and custom application events
