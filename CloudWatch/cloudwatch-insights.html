CloudWatch Insights

Log Analytics & Container Monitoring

CloudWatch Insights Overview

CloudWatch Insights includes Logs Insights for interactive log analytics and Container Insights for comprehensive container monitoring.

Logs Insights

  • • Interactive log analysis
  • • Purpose-built query language
  • • Real-time search and filter
  • • Visualization and statistics

Container Insights

  • • ECS and EKS monitoring
  • • Container performance metrics
  • • Resource utilization tracking
  • • Automated dashboards

Logs Insights Query Language

Basic Query Structure

fields @timestamp, @message
| filter @message like /ERROR/
| sort @timestamp desc
| limit 20

Statistical Analysis

fields @timestamp, @message
| filter @message like /ERROR/
| stats count() by bin(5m)
| sort @timestamp desc

Container Insights

ECS Monitoring

  • • Task and service metrics
  • • CPU and memory utilization
  • • Network performance
  • • Container lifecycle events

EKS Monitoring

  • • Pod and node metrics
  • • Kubernetes events
  • • Cluster performance
  • • Application insights

Key Takeaways

  • Logs Insights provides powerful querying capabilities for log analysis
  • Container Insights offers comprehensive monitoring for ECS and EKS
  • Purpose-built query language enables complex log analysis and visualization
