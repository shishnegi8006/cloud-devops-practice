Hybrid Connectivity

VPN & Direct Connect

Site-to-Site VPN

Encrypted connection between on-premises network and AWS VPC over the internet.

  • IPsec encrypted tunnel
  • Quick to set up (minutes)
  • Lower cost option
  • Internet-dependent performance
  • Redundant tunnels for HA

Client VPN

Managed client-based VPN service for secure remote access to AWS and on-premises resources.

Features

  • OpenVPN-based
  • Elastic scaling
  • Integration with Active Directory
  • Split-tunnel support

AWS Direct Connect

Dedicated network connection from on-premises to AWS, bypassing the internet.

Benefits

  • Consistent network performance
  • Lower bandwidth costs
  • Private connectivity
  • Up to 100 Gbps bandwidth

Connection Types

  • Dedicated Connection (1-100 Gbps)
  • Hosted Connection (50 Mbps - 10 Gbps)

VPN vs Direct Connect

Site-to-Site VPN

  • Quick setup (minutes)
  • Lower cost
  • Internet-dependent
  • Variable performance
  • Encrypted by default

Direct Connect

  • Longer setup (weeks/months)
  • Higher cost
  • Dedicated connection
  • Consistent performance
  • Not encrypted by default

Exam Strategy Tip

Remember: VPN is quick and cheap but internet-dependent. Direct Connect is dedicated and consistent but takes longer to set up. Client VPN for remote users. Use VPN + Direct Connect for backup.
