Connectivity Options

S3 Origins

  • Static website hosting and file distribution
  • Origin Access Identity for security
  • Automatic failover to secondary origins
  • Custom headers and authentication
  • Integration with S3 Transfer Acceleration

Lambda@Edge

  • Run code at AWS edge locations
  • Customize content delivery logic
  • A/B testing and personalization
  • Authentication and authorization
  • Real-time image processing

Exam Tips for Origins & Integrations

  • S3 origins work with OAI for secure content delivery
  • Custom origins support EC2, ELB, and on-premises servers
  • Lambda@Edge enables edge computing and content customization
  • Multiple origins provide redundancy and failover capabilities
