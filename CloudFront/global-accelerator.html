Global Accelerator

Network Acceleration

Anycast IP Addresses

  • Two static IP addresses for your accelerator
  • Anycast from AWS edge network
  • Route traffic to optimal AWS edge location
  • Improve performance by up to 60%
  • Automatic failover between edge locations

Traffic Dials

  • Control percentage of traffic to endpoint groups
  • Blue/green deployments and testing
  • Gradual traffic shifting for updates
  • Range from 0% to 100%
  • Real-time traffic control

Exam Tips for Global Accelerator

  • Global Accelerator uses anycast IP addresses for optimal routing
  • Traffic dials enable blue/green deployments and gradual rollouts
  • Health checks ensure traffic only goes to healthy endpoints
  • Works with TCP and UDP traffic, not just HTTP/HTTPS
