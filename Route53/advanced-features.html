Advanced Features

Resolver, DNSSEC & Traffic Flow

Route 53 Resolver

Provides recursive DNS resolution for VPCs and enables hybrid DNS resolution between AWS and on-premises.

Inbound Endpoints

On-premises DNS resolvers forward queries to Route 53

Outbound Endpoints

Forward DNS queries from VPC to on-premises DNS

DNSSEC

DNS Security Extensions provide authentication and integrity verification for DNS responses.

  • DNS response authentication
  • Data integrity verification
  • Protection against spoofing
  • Cache poisoning prevention
  • Automatic key management

DNS Firewall

Provides protection against DNS-based threats by filtering and monitoring DNS queries.

Protection Features

  • Block malicious domains
  • Filter DNS queries
  • Custom domain lists
  • AWS managed threat intelligence

Rule Actions

ALLOW, BLOCK, ALERT, Override with custom response

Traffic Flow

Visual editor for creating complex routing configurations using a drag-and-drop interface.

  • Visual policy editor
  • Complex routing logic
  • Policy versioning
  • Global traffic view
  • Geolocation rules with fallbacks
  • Multi-region failover scenarios

Exam Strategy Tip

Remember: Resolver enables hybrid DNS between AWS and on-premises. DNS Firewall protects against malicious domains. DNSSEC provides DNS response authentication. Traffic Flow offers visual policy creation.
